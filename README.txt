Tenemos dos archivos el archivo tarea TCFF.py contiene el Two-commodity formulation que es un modelo que no se logro implementar bien
el segundo archivo es tarea.py que contiene el Three-index formulation que funciona
para ejecutar tarea.py escribir:

python3 tarea.py "nombre_archivo" 

donde nombre_archivo corresponde al nombre de la instancia
En caso de que se quiera ejecutar una instancia con 40 nodos se debe agregar la string "moreNodes", entonces se ejecutaria asi:

python3 tarea.py "nombre_archivo" moreNodes

TCFF puede ejecutarse igualmente, sin embargo, las respuestas obtenidas por este son incorrectas
